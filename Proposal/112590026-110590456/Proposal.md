# Game Introduction

開啟之後會有遊戲選單，有踩地雷、接水果等遊戲可進入遊玩。
每次進入關卡，通關條件固定、畫面狀態隨機，能留下過往最高紀錄，遊戲有暫停鍵可供玩家暫停遊戲與跳出。
收集遊戲幣、解鎖關卡，展開屬於自己的像素探險！
