# Game Introduction

一場復古風的像素冒險，融合了策略與反應挑戰！
你需要在危機四伏的地雷戰場中精準推理，同時也要操控靈活的貪吃蛇吞食道具、突破極限！
收集遊戲幣、解鎖關卡，展開屬於自己的像素探險！
