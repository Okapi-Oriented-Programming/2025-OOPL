# Game Introduction

玩家使用W.A.S.D控制角色移動、滑鼠左鍵瞄準攻擊方向。開局預設普通攻擊模式，在有限時間內達到通關分數即可進入下一關，同時可以選擇一項新的能力；反之，若未在時間內達到通關分數，即為失敗。
