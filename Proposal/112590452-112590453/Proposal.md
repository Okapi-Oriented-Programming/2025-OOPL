# Game Introduction

- 一款2D縱向卷軸跑酷遊戲
- 角色：基礎動作與跳躍之外，可以蹬牆跳，上下左右、右上右下等8個方位衝刺
- 通關方式：利用地形與道具等物件，設法通往每個場景的頂端
- 場景：共1關，31個場景，中間會切換場景色調與音樂
- 影片參考：https://www.youtube.com/watch?v=Rmt_fHByXtE
