# Game Introduction
**目標：This game is inspired by the classic Flappy Bird and interesting Jetpack Joyride game, but with exciting improvements such as level-based progression and special abilities to enhance gameplay. Unlike the original endless mode, this version offers a structured stage system.

- Week 1: Concept and Asset Preparation
  - Finalize game concept and mechanics.
  - Collect or create assets (bird, pipe, background, font, sound effect, etc).

- Week 2: Initial Setup and Design
  - Create an initial game UI layout.
  - Sketch level progression ideas.
  - Set up character skin selection.

Week 3-4: Basic Game Setup
  - Implement bird movement
  - Create the initial game loop:
    - Background scrolling.
    - Pipes spawning.
  - Implement basic input handling

Week 5: Collision Detection
  - Implement collision detection:
    - Bird vs. pipes.
    - Bird vs. ground.

Week 6: Basic Gameplay Mechanics
  - Set up scoring system:
    - Increase score when passing pipes.
    - Display real-time score.
  - Implement game-over logic:
    - Show restart button.
    - Display final score.
  - Basic sound effects

Week 7: UI & Visual Effects Enhancement
  - Design UI elements:

    - Pause and Restart buttons.
  - Add sound and music:
    - Background music loop.
    - Additional sound effects for better game feel.

Week 8: Level 1
