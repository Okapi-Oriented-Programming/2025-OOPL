# Game Introduction

- 遊戲內容：操控遊戲角色瑪利歐經過層層關卡到達終點(綠色旗子處)，玩家在遊戲地圖中持續奔跑、跳躍，躲避怪物的攻擊，此外可透過道具、遊戲機制擊敗怪物及獲取金幣，目標是到達終點
- 獲勝條件：瑪利歐到達綠色旗子處
- 失敗情況：失去所有生命值
- 關卡情形：每個關卡具有不同的地圖障礙及怪物擺放位置
- 遊戲影片：(https://youtu.be/-avspZlbOWU?si=w2jzLgnK2WW6tRMu)
