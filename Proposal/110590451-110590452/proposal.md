# Game Introduction

總共會做三十個關卡，前十關是基本的推箱子遊戲，隨著關卡的推進會逐漸增加難度，第十一關開始則會開始有移動次數限制，若在限制內沒有破關就會死掉並需要重新開始遊戲，解完三十個關卡就獲勝。
