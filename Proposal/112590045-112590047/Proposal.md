# Game Introduction

玩家操作名為馬力歐的角色，通過擊殺或迴避怪物，並且通過地圖的陷阱往地圖的終點走去，為了抵達終點拯救公主。
相關介紹影片：[text](https://www.youtube.com/watch?v=Vm_w1hTgpak)
