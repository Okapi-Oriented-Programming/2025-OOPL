# Game Introduction

壞壞冰淇淋是一款躲避敵人、收集水果的遊戲，角色具有改變地形的遊戲(製造冰塊、摧毀冰塊)

遊玩影片：https://www.youtube.com/watch?v=yRM24PPbNg4

預計做出十關、五種(以上?)敵人
